A simple minesweeper app with configurable difficulty levels by changing number of rows, columns and number of mines. The
game is ended after a timeout of 5 min. User can flag a square as mine by right clicking on a square, and user can unflag
it by just clicking on a flagged square.

Tools used: Vanilla JS, Mocha as unit test framework, Chai assertions

How to run?
Just run the file /MineSweeper/index.html to play the game

How to see unit test results?
Just run the file /MineSweeper/testrunner.html
